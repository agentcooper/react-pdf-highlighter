Welcome to the Stickit Documentation!
=================================================

The `stickit <https://github.com/shlee87/public-sticky-notes-for-open-source-textbooks>`_ is a browser-based tool for making annotations on web
pages. Stick-It is a Chrome extension for annotating and discussing open-source textbooks. Users can highlight text, leave comments, and engage in conversations with others. The extension allows personalization of the learning experience by posing questions or responding to inquiries. AWS RDS is used for database management, and the backend runs on Node.jsÂ®, ensuring a scalable, secure, and efficient platform. Stick-It aims to transform interactions with open-source materials, fostering community and knowledge-sharing among learners.
